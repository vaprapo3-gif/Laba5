# Лабораторна робота №5
## Тема: С++. Проєктування та налагодження програм обробки структур

### Мета роботи
Навчитися створювати нові типи даних у вигляді структур, а також розробляти алгоритми їх обробки засобами мови C++.

---

## Умова завдання (варіант 8)
Структура містить такі поля:
- прізвище і ім’я абітурієнта;
- назва спеціальності;
- результати вступних іспитів.

Потрібно вивести на екран прізвища та назви спеціальностей для всіх абітурієнтів, що мають бал більший, ніж прохідний.

---

## Опис задачі
У програмі створено структуру `Abiturient`, яка містить дані про абітурієнта:
- `fullname` — прізвище і ім’я;
- `specialty` — назва спеціальності;
- `examResult` — результат вступного іспиту.

Програма дозволяє:
1. Ввести кількість абітурієнтів.
2. Ввести прохідний бал.
3. Ввести дані про кожного абітурієнта.
4. Перевірити, чи є абітурієнти, у яких бал більший за прохідний.
5. Вивести прізвища та назви спеціальностей тих абітурієнтів, які пройшли умову.

---

## Опис алгоритму розв’язання
1. Оголосити структуру `Abiturient`.
2. Створити масив структур для збереження даних абітурієнтів.
3. Ввести кількість абітурієнтів.
4. Ввести прохідний бал.
5. У циклі ввести дані для кожного абітурієнта:
   - прізвище та ім’я;
   - назву спеціальності;
   - результат вступного іспиту.
6. Переглянути всі записи масиву.
7. Якщо результат вступного іспиту абітурієнта більший за прохідний бал, вивести його прізвище та назву спеціальності.
8. Якщо таких абітурієнтів немає, вивести повідомлення про це.

---

## Пояснення основних етапів роботи програми

### 1. Опис структури
Структура `Abiturient` використовується для збереження даних про одного абітурієнта.

### 2. Введення даних
Користувач вводить кількість абітурієнтів, прохідний бал та інформацію про кожного абітурієнта.

### 3. Обробка даних
Програма перевіряє результат вступного іспиту кожного абітурієнта.

### 4. Виведення результату
На екран виводяться тільки ті абітурієнти, у яких бал більший за прохідний.
